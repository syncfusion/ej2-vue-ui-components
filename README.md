# Syncfusion Vue UI Components Library – Essential JS 2

Syncfusion Essential JS 2 (EJ2) for Vue is a complete enterprise-grade UI component library with **145+** high-performance, responsive, lightweight, modular, and touch-friendly Vue components. Build modern, cross-platform web applications faster with rich features like accessibility (WCAG 2.2 compliant), built-in themes (Material 3, Bootstrap 5.3, Tailwind3, Fluent2, etc.), globalization, right-to-left (RTL) support, and seamless Vue integration.

## Features

## Features

- **145+ Components** — Full suite built natively for Vue (no wrappers needed), supporting Vue 3 Composition API and Options API.
- **AI-Powered Smart Components** — AI AssistView, Smart TextArea, Smart Paste Button, and AI integrations in DataGrid, Spreadsheet, and more.
- **Vue 3 Compatibility** — Full support for Vue 3 (including latest versions); dedicated tutorials for Vue 3 setup.
- Regular updates: 4+ major releases per year + monthly patches.
- Modular installation — Install only the components you need to keep bundle size minimal.
- Enterprise-grade features — Accessibility, internationalization, RTL, theming (Material 3, Bootstrap 5, Tailwind, Fluent, etc.), and high performance with virtualization.
